# Stable Diffusion v1.5 本地部署说明

## 环境

本项目使用独立虚拟环境 `.venv`，避免污染系统或 conda base 环境。

创建环境：

```bash
uv venv --python python3.12 .venv
```

安装依赖：

```bash
uv pip install --python .venv/bin/python -r requirements.txt
```

## Hugging Face 镜像

生成脚本默认设置：

```bash
HF_ENDPOINT=https://hf-mirror.com
HF_HOME=.cache/huggingface
```

模型会缓存到项目目录下的 `.cache/huggingface`，该目录不会提交到 Git。

## 生成测试

默认测试命令：

```bash
.venv/bin/python scripts/generate_sd15.py \
  --output-dir outputs/sd15_base_test \
  --seeds 42 \
  --steps 20
```

离线缓存验证：

```bash
.venv/bin/python scripts/generate_sd15.py \
  --output-dir outputs/sd15_offline_check \
  --prompt "watercolor style, a red umbrella on a quiet street" \
  --seeds 7 \
  --steps 10 \
  --local-files-only
```
