# Watercolor Legacy Archive

这里保存旧的 Stable Diffusion v1.5 水彩风景 LoRA 方案。它不是当前项目路线。

归档内容包括：

- 旧中文规划文档：`docs/`
- 旧 Beamer 汇报和图片素材：`slides/`、`assets/beamer/`
- 旧 SD1.5 base 生成脚本：`scripts/generate_sd15.py`
- 旧根目录示例图片：`data/001.png` 到 `data/004.png`

当前项目请看仓库根目录 [README.md](../../README.md)，不要把这里的 `watercolor style` prompt、SD1.5 分辨率和旧数据结构套到 SDXL 动漫头像 LoRA 实验上。
